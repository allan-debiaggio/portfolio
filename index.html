Projets en cours

Rien à déclarer ?

En tant que développeur logiciel, le projet dont je suis le plus fier pour le moment est un jeu de Morpion qui se trouve sur mon GitHub (lien plus bas), dans le repository TicTacToe. Rien de très visuellement attrayant, seul un modeste programme qui tourne sur Visual Studio Code. Vous pouvez évidemment télécharger le fichier final se trouvant sur la branche "main" pour l'essayer dans votre VSCode. Etant donné la qualité peu originale de celui-ci (projet commun à tous les étudiants en première année dans leur semaine d'essai Python), il est recommandé de jeter un coup d'oeil régulier sur mon GitHub pour voir arriver d'autres projets plus atypiques en lien direct avec la spécialisation de Développeur Logiciel qui, je l'espère, me sera attribuée d'ici environ un mois. Il n'est pas exclu, voire très probable que j'y poste également des projets personnels en lien avec mes lectures et tutoriaux du moment, dès que j'ai du temps libre.
